Research Background

The explosive growth of data, which we have been witnessing since the onset of the new century, has greatly reshaped our everyday life. From oil wells to shopping malls, from New York to Hong Kong, massive amount of data is being acquired every hour, and thus analyzing large data sets (often called big data) has become the most important challenge that we face as computer scientists. In alignment with this trend, my research centers on the development of probabilistic models and inference techniques to analyze large-scale data, especially those arising from computer vision, text mining, and related areas.

Below is a selected list of research projects that I have done in recent year, each with a brief description.

  • Large-scale PLC Architecture Design

    Description

    Leader of subproject of 863 plan. Carried out a survey of mainstream PLC manufacturer’ architecture solutions, We prefer Heterogeneous System Architecture based on Xilinx zynq, which includes an ARM dual-core cpu and a high capacity FPGA onboard. We succeed to port and run our RTOS running on ARM and implemented a PCI express IP running on FPGA alongside. Meanwhile, we designed a new I/O module with PCI express interface support. Corresponding peripheral drivers and applications to test the system bandwidth and RTOS benchmarks were finished successfully. We have achieved bandwidth of 12.67Gib/s during actual measurement.

  • SOCOS: real-time operating system based on Xilinx SOC technology

    Description

    This is a real-time operating system running on Xilinx zynq-7000 SOC. Current OS depends on on-chip timer interrupt to kernel mode and running scheduling logic. To minimize the OS loss, I wrote the SOCOS, whose scheduling function is running by parallel hardware logic, such as FPGA. As so far, I have completed the boot-loader, memory manager (Buddy and slab algorithm), round-robin scheduling and so on. Hardware scheduling module remains unfinished.
